# loader 列表

## 处理css

1. css-loader: 将 CSS 等价翻译为 `module.exports = "${css}"` 的JavaScript 代码
2. style-loader: 在产物中注入一系列 runtime 代码，这些代码会将 CSS 内容注入到页面的 `<style>` 标签
3. mini-css-extract-plugin: 将 CSS 代码抽离到单独的 .css 文件，并将文件通过 `<link>` 标签方式插入到页面中。
