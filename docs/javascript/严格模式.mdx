# 严格模式

## 出现的目的
在ECMAScript 5增加，用于处理 ECMAScript 3的一些不规范写法，对于不安全的活动将抛出错误

```js title=
  "use strict";
```
任何支持的JavaScript引擎看到它都会切换到严格模式。

## 注意事项

1. 严格模式下不能将`eval`和`arguments`为定义为变量。
2. 严格模式下可以定义`undefined`为变量。
3. 严格模式下函数的参数不能叫`eval`或`arguments`

```js
const undefined = 10
```

3. 在严格模式下，如果给未声明的变量赋值，则会导致抛出ReferenceError。

```js title=
message = 'hello';
```
